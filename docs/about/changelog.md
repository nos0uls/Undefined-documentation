# Changelog

Отдельный ручной changelog в этом docs-репозитории сейчас не ведётся.

## Где смотреть изменения

- история git-коммитов репозитория
- diff по документации в pull request
- разделы `planning/reviews`, если нужен контекст по крупным итерациям

## Практическая рекомендация

Если нужно понять, когда и почему менялся конкретный документ, надёжнее смотреть историю файла в git, а не ждать отдельную сводку на этой странице.
