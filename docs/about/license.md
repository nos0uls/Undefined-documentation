# License

На этой странице пока нет отдельного зафиксированного текста лицензии для документации.

## Что это означает на практике

- перед переиспользованием материалов лучше уточнить условия у владельца репозитория
- наличие исходников и документации в GitHub само по себе не заменяет явную лицензию
- лицензии сторонних библиотек и инструментов нужно смотреть отдельно в их собственных репозиториях

## Если нужна формальная лицензия

Лучше добавить её как отдельный canonical файл в репозиторий и затем сослаться на него из этой страницы.
