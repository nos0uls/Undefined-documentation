# Настройка окружения (Setup Guide)

Добро пожаловать в проект **Undefinedtale-888**! Это руководство поможет вам настроить рабочее окружение с нуля.

## 1. Установка GameMaker Studio 2

Проект разработан на **GameMaker Studio 2** (версия 2023+ или новее).
1.  Скачайте и установите GMS2 с [официального сайта](https://gamemaker.io/en/get).
2.  Убедитесь, что у вас есть активная лицензия (Free, Indie или Professional).
3.  Войдите в свой аккаунт YoYoGames при запуске.

## 2. Зависимости (Dependencies)

В проекте используются сторонние библиотеки для GMS2. Они уже включены в репозиторий, но полезно знать, что это:
-   **[Scribble](https://github.com/JujuAdams/Scribble)**: Мощная система рендеринга текста.
-   **[Chatterbox](https://github.com/JujuAdams/Chatterbox)**: Парсер файлов Yarn для диалогов.
-   **[Input](https://github.com/JujuAdams/Input)**: (Опционально, если используется) Система ввода.

!!! tip "Примечание"
    Вам **не нужно** скачивать их отдельно. Они находятся в папке `scripts/` и готовы к работе.

## 3. Первый Запуск

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/n0souls/Undefinedtale-888.git
    ```
2.  Откройте файл проекта `Undefinedtale888.yyp` в GameMaker Studio 2.
3.  Дождитесь индексации ассетов.
4.  Нажмите **F5** (или кнопку Play), чтобы запустить проект.

## 4. Настройка IDE (Рекомендации)

Для удобной работы с кодом GML:
-   Используйте **JSDoc** комментарии для функций (`/// @desc`, `/// @param`).
-   Соблюдайте стиль именования:
    -   Объекты: `obj_Name`
    -   Спрайты: `spr_Name`
    -   Скрипты: `scr_camelCase`
    -   Переменные: `snake_case`

## 5. Документация

Локальная документация (этот сайт) запускается командой:
```bash
# В папке my-docs-repo
mkdocs serve
```
Откройте `http://127.0.0.1:8000` в браузере.
