# Первый Запуск (First Run)

Поздравляем с успешной настройкой проекта! Вот что вас ждет при первом запуске.

## Запуск из IDE

1.  В GameMaker Studio 2 нажмите `F5` или кнопку **Play** на панели инструментов.
2.  Начнется компиляция проекта.
3.  Откроется окно игры.

## Ожидаемое поведение

1.  **Черный экран (1-2 сек)**: Инициализация систем в `rm_init`.
2.  **Главное Меню**: Вы должны увидеть титульный экран с пунктами меню.
3.  **Музыка**: Должна заиграть фоновая музыка меню.

## Управление в меню

*   **Стрелки / WASD**: Навигация.
*   **Z / Enter / Пробел**: Подтверждение.
*   **X / Esc**: Отмена / Назад.

## Решение проблем

Если игра вылетает при запуске:
*   Проверьте `Output` окно в IDE на наличие ошибок.
*   Убедитесь, что файл `game_state.dat` не поврежден (попробуйте удалить его, игра создаст новый).
