# Комнаты (Rooms)

Список основных комнат проекта и их назначение.

| Комната | Назначение |
|---------|------------|
| `rm_init` | **Точка входа**. Пустая комната, где происходит только инициализация (`obj_Init`). Сразу переключает на меню. |
| `rm_roomMenu` | **Главное меню**. |
| `rm_savesSelect` | Экран выбора сохранения. |
| `rm_settings` | Меню настроек (если реализовано как комната). |
| `rm_devLoad` | **DEV-LOAD**. Техническая комната для быстрого перехода на уровни при разработке. |
| `rm_*` | Игровые уровни (школа, город и т.д.). |

## Организация

Комнаты загружаются по имени через глобальную карту `global.rooms_by_name`, что позволяет избегать жестких ссылок на asset index при загрузке сохранений.
