# Объекты Системы (System Objects)

В проекте используется несколько ключевых персистентных объектов, которые управляют глобальной логикой.

| Объект | Persistent | Назначение |
|--------|-----------|------------|
| `obj_Init` | Нет | **Центральная инициализация**. Создается один раз при старте, настраивает всё и исчезает (переходя в меню). |
| `obj_globalManager` | Да | **Runtime-менеджер**. Живет всю игру. Отвечает за музыку, уведомления, смену комнат и дебаг-функции. |
| `obj_menu` | Нет | Главное меню игры. |
| `obj_settingsManager` | Нет | Управляет настройками (Overlay или отдельная комната). |
| `obj_saveManager` | Нет | Экран выбора слота сохранения и DEV-LOAD. |
| `obj_inGameMenu` | Нет | Внутриигровое меню (Инвентарь, Статус, Настройки). |
| `obj_player` | Нет | Игровой персонаж. |
| `obj_actor` | Нет | Базовый объект для NPC и участников катсцен. |
| `obj_pointMarker` | Нет | Маркер взаимодействия перед игроком. |
