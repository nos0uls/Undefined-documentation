# Система Музыки (Music System)

В проекте используется централизованная система управления музыкой, чтобы избежать хардкода в каждом объекте.

## Как это работает?

### 1. Карта Музыки (Music Map)
В файле инициализации `obj_Init` создается глобальная карта `global.room_music_map`. Она связывает название комнаты с аудио-ресурсом.

```gml
// obj_Init -> Create
global.room_music_map = {};
global.room_music_map[$ room_get_name(rm_school_corridor)] = music_SchoolRoutine;
global.room_music_map[$ room_get_name(rm_roomMenu)] = music_menu;
```

### 2. Смена Комнаты
Когда игрок переходит в другую комнату, скрипт `scr_global_on_room_change` автоматически проверяет эту карту.
*   Если комната есть в карте -> играет указанная музыка.
*   Если комнаты нет -> музыка не меняется (продолжает играть старая).

### 3. Плавные Переходы
`obj_globalManager` отвечает за плавное затухание (fade-out) старого трека и появление (fade-in) нового.
*   `global.play_music(snd)`: Плавная смена.
*   `global.play_music_immediate(snd)`: Мгновенная смена (резкая).

## Как добавить музыку в новую комнату?
1. Откройте `obj_Init`.
2. Найдите секцию `// 3.1. Карта музыки для комнат`.
3. Добавьте строчку:
   ```gml
   global.room_music_map[$ room_get_name(ВАША_КОМНАТА)] = ВАШ_ТРЕК;
   ```
