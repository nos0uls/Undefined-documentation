# Теги

Список тегов, используемых в документации. Теги помогают группировать страницы по темам и быстро находить связанный контент.

<!-- material/tags -->

---

!!! tip "Как добавить теги"
    Чтобы добавить теги к странице, укажите их в frontmatter:

    ```yaml
    ---
    tags:
      - инициализация
      - глобальное-состояние
    ---
    ```
