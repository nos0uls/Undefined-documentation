# Undefscene: Форматы, импорт и экспорт

## Основные форматы

### `.usc.json`

Это рабочий формат редактора.

Он хранит:

- граф катсцены
- позиции нод
- editor-only данные
- layout и вспомогательное состояние, нужное для продолжения работы

Используйте этот формат для сохранения проекта, который вы будете продолжать редактировать.

### Engine `.json`

Это runtime-экспорт для движка GameMaker.

Он содержит:

- массив `actions`
- параметры команд, нужные движку
- минимум editor-only информации

Используйте этот формат для передачи катсцены в игру.

## Сохранение и загрузка

- **Save Scene / Save As...** — сохраняет рабочую сцену как `.usc.json`.
- **Open Scene** — открывает `.usc.json` или поддерживаемый `.json` обратно в редактор.
- **New Scene** — создаёт новую пустую сцену.
- **Create Example** — загружает демонстрационную сцену.

## Export for Engine

Команда **Export for Engine** генерирует чистый `.json`, пригодный для использования в проекте GameMaker.

Такой файл обычно сохраняют в папку `datafiles` проекта, рядом с другими runtime-ресурсами игры.

## Reverse import

Редактор умеет открывать поддерживаемые engine `.json` файлы обратно.

При таком reverse-import редактор восстанавливает граф сцены из массива `actions`, если структура файла соответствует ожидаемому engine-формату.
Если структура файла повреждена, неполная или неоднозначна, импорт отклоняется как unsupported format, чтобы не строить неверный граф.
