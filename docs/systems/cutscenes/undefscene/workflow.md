# Undefscene: Создание катсцены

Пошаговый процесс создания катсцены от начала до экспорта.

## Шаг 1: Подготовка

1. Запустите редактор
2. Откройте проект GameMaker: **Open Project** → выберите файл `.yyp`
3. Это включит автокомплит имён объектов, спрайтов и диалогов

## Шаг 2: Новая сцена

Нажмите **New Scene**. На холсте появится нода **Start** — с неё начинается любая катсцена.

## Шаг 3: Добавьте ноды

Перетащите нужные ноды из палитры слева на холст. Или кликните средней кнопкой по пустому месту (создаётся Dialogue).

Типичная катсцена выглядит так:

```
Start → Actor Create → Move → Dialogue → Actor Destroy → End
```

## Шаг 4: Соедините ноды

Потяните от правого края одной ноды (handle) к левому краю другой. Появится линия (ребро) — по ней катсцена переходит от одной ноды к следующей.

## Шаг 5: Настройте параметры

Кликните по ноде → в правой панели (Inspector) заполните параметры. Обязательные поля отмечены — без них редактор покажет предупреждение.

## Шаг 6: Добавьте задержки

Если между двумя действиями нужна пауза — двойной клик по ребру (линии) и введите секунды. Это проще, чем вставлять отдельную Wait-ноду.

## Шаг 7: Проверьте ошибки

Откройте панель **Логи / Предупреждения**. Если есть ошибки или предупреждения — исправьте их перед экспортом.

Частые проблемы:
- Пустое поле Target у ноды движения
- Не выбран .yarn файл у Dialogue
- Нода End не соединена с графом

## Шаг 8: Сохраните и экспортируйте

1. **Save** — сохраняет рабочую сцену (`.usc.json`) для продолжения редактирования
2. **Export for Engine** — создаёт файл для игры (`.json`), сохраните его в `datafiles/cutscenes/` проекта

!!! warning "Не путайте"
    **Save** ≠ **Export for Engine**. Save — для редактора, Export — для игры.

## Шаг 9: Используйте в игре

В коде GameMaker вызовите:

```gml
var _mgr = cutscene_load_json("cutscenes/my_scene.json");
_mgr.start_cutscene();
```

---

## Советы

- Сначала откройте `.yyp`, потом настраивайте параметры — так работает автокомплит
- Для сложных маршрутов используйте **Visual Editing** — рисуйте путь на фоне комнаты
- Для Dialogue-нод проверяйте текст через **Yarn Preview** в панели Text
- Перед экспортом всегда проверяйте панель **Логи / Предупреждения**
