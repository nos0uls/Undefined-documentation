# Undefscene: Глоссарий

Краткие определения терминов, которые встречаются в документации.

| Термин | Определение |
|--------|------------|
| **Нода (Node)** | Блок на холсте — одно действие в катсцене (движение, диалог, камера и т.д.) |
| **Ребро (Edge)** | Линия между нодами — переход от одного действия к следующему |
| **Inspector** | Правая панель, где настраиваются параметры выбранной ноды |
| **Target** | Цель действия — ключ актёра (например, `npc_guide`) или `player` |
| **Key** | Уникальное имя актёра. Задаётся в Actor Create, используется в других нодах как Target |
| **Actor** | Персонаж или объект на сцене катсцены. Создаётся через Actor Create |
| **Yarn** | Формат диалоговых файлов (`.yarn`). Используется движком Chatterbox |
| **.yyp** | Файл проекта GameMaker. Открытие `.yyp` в редакторе включает автокомплит и проверки |
| **.usc.json** | Рабочий формат редактора. Хранит граф, позиции нод, состояние панелей |
| **Engine .json** | Файл для игры. Создаётся через Export for Engine |
| **Export for Engine** | Команда экспорта катсцены в формат, который игра может прочитать |
| **Visual Editing** | Отдельное окно с фоном комнаты для расстановки актёров и рисования путей |
| **Yarn Preview** | Панель с текстом диалога из `.yarn` файла — можно прочитать, не запуская игру |
| **Path Preview** | Превью траектории в Inspector для Follow Path ноды |
| **Edge Condition** | Условие на ребре — переход произойдёт только если условие истинно |
| **Wait on Edge** | Задержка на ребре — пауза перед переходом к следующей ноде |
| **Parallel** | Параллельное выполнение — несколько действий одновременно |
| **Branch** | Ветвление — разные пути в зависимости от условия |
| **Easing** | Тип плавности анимации: `linear`, `ease_in`, `ease_out`, `ease_in_out` |
| **Sprite** | Картинка-спрайт из проекта GameMaker |
| **Depth** | Слой глубины. Меньше значение = ближе к камере |
| **Facing** | Направление, куда смотрит актёр: left, right, up, down |
| **Emote** | Маленькая картинка над актёром (восклицательный знак, сердечки и т.п.) |
| **Tween** | Плавное изменение свойства со временем (например, прозрачности) |
| **SFX** | Звуковой эффект (Sound Effect) |
| **Fade In / Fade Out** | Плавное появление / затемнение экрана |
| **Instant Mode** | Режим, где все действия выполняются мгновенно (для отладки) |
| **Mark Node** | Именованная метка в графе — на неё можно ссылаться из условий |
| **Halt** | Остановка актёра и блокировка дальнейшего выполнения |
| **Autosave** | Автоматическое сохранение сцены через заданный интервал |
