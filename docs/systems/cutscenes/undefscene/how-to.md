# How-to: частые задачи

Короткие рецепты для типичных задач в Undefscene. Каждый рецепт — 3–5 шагов.

---

## Как создать первую катсцену

1. Откройте редактор → **New Scene**
2. Добавьте **Start** ноду (она уже есть на холсте)
3. Перетащите из палитры **Dialogue** ноду, соедините со Start
4. В Inspector выберите `.yarn` файл и ноду диалога
5. Добавьте **End** ноду после Dialogue, соедините
6. **Export for Engine** → сохраните `.json` в `datafiles` проекта

---

## Как добавить диалог

1. Перетащите **Dialogue** ноду из палитры на холст
2. В Inspector в поле **File** выберите `.yarn` файл
3. В поле **Node** выберите нужную ноду диалога
4. Соедините Dialogue с предыдущей нодой ребром
5. Проверьте текст в **Yarn Preview** (панель Text)

---

## Как заставить актёра идти по пути

1. Добавьте **Actor Create** ноду — задайте key, координаты, спрайт
2. Добавьте **Follow Path** ноду — в поле Target укажите key актёра
3. Откройте **Visual Editing** (кнопка на панели)
4. Нарисуйте путь инструментом **Draw Path** (или **Pencil** для свободного рисования)
5. Нажмите **Import Path** — координаты автоматически попадут в ноду

!!! tip "Shift и Ctrl в Visual Editing"
    - **Shift** — привязка к горизонтали/вертикали
    - **Ctrl** — привязка к сетке 20px
    - **Shift + Ctrl** — оба режима сразу

## Как настроить камеру

=== "Плавное перемещение в точку"

    1. Добавьте **Camera Pan** ноду
    2. Укажите координаты X, Y и длительность в секундах
    3. Поставьте ноду перед действиями, которые камера должна показать

=== "Следить за актёром"

    1. Добавьте **Camera Track** ноду
    2. Укажите Target (ключ актёра) и длительность
    3. Камера будет следовать за актёром указанное время

=== "Тряска камеры"

    1. Добавьте **Camera Shake** ноду
    2. Укажите длительность и силу тряски (magnitude)
    3. По умолчанию magnitude = 4 px

---

## Как создать параллельные действия

Параллельные действия выполняются одновременно — например, камера едет, а NPC одновременно идёт.

1. Добавьте **Parallel Start** — от него пойдут две ветки
2. Создайте первую цепочку действий (например, Camera Pan)
3. Создайте вторую цепочку (например, Move для NPC)
4. Обе ветки должны сойтись в **Parallel Join**
5. Редактор автоматически создаёт пару Start + Join

## Как добавить условие перехода

=== "Через Branch-ноду"

    1. Добавьте **Branch** ноду
    2. В Inspector выберите условие из списка
    3. От Branch идут два ребра: одно для `true`, другое для `false`

=== "Через Edge Condition"

    1. Выберите ребро (линию между нодами)
    2. Включите чекбокс **Condition** в Inspector
    3. Укажите **Variable** (имя глобальной переменной) и **Equals** (ожидаемое значение)
    4. Выберите **If false**: `skip` (пропустить) или `wait until true` (ждать)

---

## Как экспортировать катсцену в игру

1. Проверьте панель **Логи / Предупреждения** — исправьте ошибки
2. Нажмите **Export for Engine**
3. Сохраните файл в папку `datafiles/cutscenes/` проекта GameMaker
4. В игре вызовите: `cutscene_load_json("cutscenes/имя_файла.json")`

!!! warning "Не путайте Save и Export"
    - **Save** — сохраняет рабочую сцену (`.usc.json`) для продолжения редактирования
    - **Export for Engine** — создаёт файл для игры (`.json`)

---

## Как добавить тряску камеры

1. Добавьте **Camera Shake** ноду
2. Укажите **Duration** (в секундах, например 1)
3. Укажите **Magnitude** (сила в пикселях, например 4)
4. Поставьте ноду в нужное место графа

---

## Как заставить актёра повернуться

=== "В конкретную сторону"

    Добавьте **Set Facing** ноду → Target + Direction (`left`/`right`/`up`/`down`)

=== "Автоматически к игроку"

    Добавьте **Auto Facing** ноду → Target + `true`. Актёр будет всегда смотреть на игрока.

---

## Как добавить звуковой эффект

1. Добавьте **Play SFX** ноду
2. В поле **Sound / Key** укажите имя звука
3. При желании настройте **Volume** (0–1) и **Pitch** (1.0 = нормальный)
