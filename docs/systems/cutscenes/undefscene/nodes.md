# Справочник нод

Катсцена в Undefscene — это граф из нод, соединённых рёбрами. Каждая нода — одно действие: движение, диалог, камера, ожидание и т.д.

Ноды группируются по цвету на холсте:

| Категория | Цвет | Ноды |
|-----------|------|------|
| Поток | синий | Start, End, Wait |
| Движение | фиолетовый | Move, Follow Path, Set Position, Jump, Halt |
| Актёры | фиолетовый | Actor Create, Actor Destroy |
| Визуал | фиолетовый | Animate, Set Facing, Set Depth, Auto Facing, Auto Walk, Flip, Spin, Shake Object, Set Visible, Emote |
| Диалог | розовый | Dialogue, Wait for Dialogue |
| Камера | зелёный | Camera Pan, Camera Pan To Object, Camera Center, Camera Track, Camera Track Until Stop, Camera Shake, Tween, Tween Camera, Fade In, Fade Out |
| Логика | оранжевый | Parallel Start, Parallel Join, Branch, Run Function, Instant Mode, Mark Node |
| Звук | бирюзовый | Play SFX |

!!! tip "Общие параметры"
    Многие ноды имеют параметр **Target** — это ключ актёра (например, `npc_guide`) или `player`. Если редактор загрузил `.yyp` проект, поле Target предлагает автокомплит из списка объектов.

## Поток (Flow)

### Start

Начальная нода катсцены. С неё начинается выполнение.

В графе должна быть ровно одна нода Start. У неё нет входящих связей.

*Параметров нет.*

### End

Конечная нода. Когда выполнение доходит до End, катсцена завершается.

В графе должна быть хотя бы одна нода End. У неё нет исходящих связей.

*Параметров нет.*

### Wait

Просто ждёт указанное количество секунд, потом переходит к следующей ноде.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| seconds | number | да | Сколько секунд ждать |

!!! tip "Wait на ребре"
    Вместо отдельной ноды Wait можно задать задержку прямо на ребре (линии между нодами). Двойной клик по линии → ввести секунды. Это удобнее, чем вставлять Wait-ноду между каждым шагом.

## Движение (Movement)

### Move

Перемещает актёра в указанную точку. Актёр идёт «пешком» — со спрайтом ходьбы, если включён Auto Walk.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра или `player` |
| x | number | да | Целевая координата X |
| y | number | да | Целевая координата Y |
| speed | number | да | Скорость движения (px/sec) |
| useCollision | bool | нет | Учитывать стены при движении |

**Пример**: NPC идёт к двери — target: `npc_guide`, x: 400, y: 200, speed: 120

### Set Position

Мгновенно перемещает актёра в точку. Никакой анимации ходьбы — актёр просто появляется в новом месте.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра или `player` |
| x | number | да | Координата X |
| y | number | да | Координата Y |

**Пример**: Спрятать актёра за экраном — target: `npc_shadow`, x: -100, y: 300

### Follow Path

Актёр идёт по маршруту из нескольких точек. Это удобнее, чем ставить несколько Move-нод подряд.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра или `player` |
| path | string | да | Маршрут в формате `x1,y1;x2,y2;x3,y3` |
| speed | number | да | Скорость движения (px/sec) |
| useCollision | bool | нет | Учитывать стены |

!!! tip "Visual Editing"
    Не обязательно писать координаты вручную. Откройте **Visual Editing** (кнопка на панели) — там можно нарисовать путь мышкой на фоне комнаты и импортировать его в ноду кнопкой **Import Path**.

### Jump

Актёр «подпрыгивает» к точке по дуге. Полезно для визуальных эффектов.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра или `player` |
| x | number | да | Целевая X |
| y | number | да | Целевая Y |
| seconds | number | да | Длительность прыжка |
| height | number | нет | Высота дуги |
| easing | select | нет | `linear`, `ease_in`, `ease_out`, `ease_in_out` |

### Halt

Останавливает актёра. Прерывает текущее движение, ходьбу, анимацию.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра или `player` |

!!! warning "Halt блокирует выполнение"
    Halt останавливает **всё**, включая очередь катсцены. Ноды после Halt не выполнятся, пока актёра не «отпустят». Не ставьте после Halt другие ноды — они не запустятся.

## Актёры (Actor)

### Actor Create

Создаёт нового актёра на сцене. Это может быть NPC, объект или любой персонаж, которого изначально нет в комнате.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| key | string | да | Уникальный ключ актёра (например, `npc_merchant`). Другие ноды будут ссылаться на него по этому ключу |
| x | number | да | Координата появления X |
| y | number | да | Координата появления Y |
| sprite | string | нет | Спрайт актёра (если не указан — используется объект по умолчанию) |
| copyFrom | string | нет | Ключ другого актёра, чей внешний вид нужно скопировать |

**Пример**: Появление торговца — key: `npc_merchant`, x: 300, y: 200, sprite: `spr_merchant_idle`

### Actor Destroy

Удаляет актёра со сцены.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра, которого нужно удалить |

## Визуал (Visual)

### Animate

Меняет спрайт актёра. Например, переключает с idle на анимацию жеста.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра или `player` |
| sprite | string | да | Новый спрайт (автокомплит из проекта) |
| image_index | number | нет | Начальный кадр (по умолчанию 0) |
| image_speed | number | нет | Скорость анимации (по умолчанию 0 — статичный кадр) |

**Пример**: NPC показывает жест — target: `npc_guide`, sprite: `spr_guide_wave`, image_speed: 0.3

### Set Facing

Поворачивает актёра в указанном направлении.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра или `player` |
| direction | select | да | `left`, `right`, `up`, `down` |

### Set Depth

Меняет глубину (слой) актёра. Чем меньше значение — тем ближе к камере.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра или `player` |
| depth | number | да | Значение глубины (например, -100 для переднего плана) |

### Auto Facing

Включает или выключает автоматический поворот актёра к игроку. Когда включено — актёр всегда смотрит в сторону игрока.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра или `player` |
| enabled | select | да | `true` / `false` |

### Auto Walk

Включает или выключает автоматическую анимацию ходьбы. Когда включено — актёр играет спрайт ходьбы при движении.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра или `player` |
| enabled | select | да | `true` / `false` |

### Flip

Отзеркаливает актёра по горизонтали.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра или `player` |
| flipped | select | да | `true` (отзеркален) / `false` (нормально) |

### Spin

Вращает актёра вокруг своей оси.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра или `player` |
| speed | number | да | Скорость вращения |
| seconds | number | да | Длительность |

### Shake Object

Трясёт актёра на месте. Полезно для эффектов удара, страха и т.п.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра или `player` |
| seconds | number | да | Длительность тряски |
| magnitude | number | нет | Сила тряски в пикселях |

### Set Visible

Показывает или прячет актёра.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра или `player` |
| visible | select | да | `true` (видим) / `false` (скрыт) |

### Emote

Показывает над актёром эмоцию — маленькую картинку (восклицательный знак, сердечки и т.п.).

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра или `player` |
| sprite | string | да | Спрайт эмоции |
| seconds | number | нет | Сколько показывать |
| offset_x | number | нет | Смещение по X от актёра |
| offset_y | number | нет | Смещение по Y от актёра |
| scale | number | нет | Масштаб эмоции |
| wait | select | нет | `true` — ждать пока эмоция закончится, `false` — показать и идти дальше |

## Диалог (Dialogue)

### Dialogue

Запускает диалог из Yarn-файла. Диалог отображается в текстовом окне игры.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| file | string | да | Имя `.yarn` файла (автокомплит из проекта) |
| node | string | да | Имя ноды внутри файла (выбор из списка после выбора файла) |

!!! tip "Yarn Preview"
    Когда выбран `.yarn` файл, в панели **Text** появляется **Yarn Preview** — можно прочитать текст диалога прямо в редакторе, не запуская игру.

### Wait for Dialogue

Ждёт, пока текущий диалог не закончится. Полезно, если после диалога нужно сразу запустить следующее действие.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| dialogue_controller | string | нет | Ссылка на конкретный диалог (если пусто — ждёт активный) |

## Камера (Camera)

### Camera Pan

Плавно перемещает камеру в указанную точку.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| x | number | да | Целевая координата X |
| y | number | да | Целевая координата Y |
| seconds | number | да | Длительность перемещения |

### Camera Pan To Object

Плавно перемещает камеру к актёру.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра, к которому едет камера |
| seconds | number | да | Длительность |

### Camera Center

Мгновенно ставит камеру в указанную точку. Без плавного перемещения.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| x | number | да | Координата X |
| y | number | да | Координата Y |

### Camera Track

Камера следует за актёром указанное время.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра, за которым следует камера |
| seconds | number | да | Длительность слежения |
| offset_x | number | нет | Смещение камеры по X |
| offset_y | number | нет | Смещение камеры по Y |

### Camera Track Until Stop

Камера следует за актёром, пока не будет остановлена вручную (другой нодой камеры).

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| target | string | да | Ключ актёра |
| offset_x | number | нет | Смещение по X |
| offset_y | number | нет | Смещение по Y |

### Camera Shake

Трясёт камеру. Классический эффект для драматических моментов.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| seconds | number | да | Длительность тряски |
| magnitude | number | нет | Сила тряски в пикселях (по умолчанию 4) |

### Tween

Плавно меняет свойство актёра со временем. Например, можно плавно изменить масштаб, прозрачность или координату.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| kind | select | да | `instance` (свойство актёра) или `camera` (свойство камеры) |
| target | string | да* | Ключ актёра (только если kind = instance) |
| property | string | да | Имя свойства (например, `image_xscale`, `image_alpha`) |
| to | number | да | Целевое значение |
| from | number | нет | Начальное значение (если не указано — берётся текущее) |
| seconds | number | да | Длительность перехода |
| easing | select | нет | `linear`, `ease_in`, `ease_out`, `ease_in_out` |

### Tween Camera

То же, что Tween с kind = camera, но как отдельная нода. Плавно меняет свойство камеры.

Параметры те же, что у Tween, но без kind и target.

### Fade In

Плавно убирает затемнение экрана (от чёрного к картинке).

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| seconds | number | да | Длительность |
| color | string | нет | Цвет затемнения (по умолчанию `black`) |

### Fade Out

Плавно затемняет экран (от картинки к чёрному).

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| seconds | number | да | Длительность |
| color | string | нет | Цвет (по умолчанию `black`) |

## Логика (Logic)

### Parallel Start / Parallel Join

Позволяет запускать несколько действий одновременно. Parallel Start — точка разветвления, Parallel Join — точка объединения.

!!! example "Пример: параллельные действия"
    Пока камера едет к точке, NPC одновременно идёт к двери. Для этого:
    
    1. Поставьте **Parallel Start**
    2. От него проведите два ребра к двум разным цепочкам действий
    3. Обе цепочки должны заканчиваться в одном **Parallel Join**

*Parallel Start и Parallel Join всегда идут парой. Редактор автоматически создаёт обе ноды при добавлении Parallel.*

### Branch

Разветвление по условию. Если условие истинно — идём по одному пути, если нет — по другому.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| condition | string | да | Имя условия (автокомплит из настроек движка) |

!!! tip "Условия на рёбрах"
    Вместо Branch можно использовать **Edge Condition** — условие прямо на линии между нодами. Выберите ребро → включите Condition → укажите переменную и значение.

### Run Function

Вызывает произвольную функцию GML. Для продвинутых пользователей.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| function | string | да | Имя функции (автокомплит из настроек движка) |
| args | string | нет | Аргументы в формате JSON, например `["arg1", 42]` |

### Instant Mode

Включает «мгновенный» режим — все последующие действия выполняются без задержки (0 кадров вместо указанного времени). Полезно для отладки.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| enabled | select | да | `true` (включить мгновенный режим) / `false` (вернуть нормальный) |

### Mark Node

Ставит именованную метку в графе. На метку можно ссылаться из условий Edge Condition (режим `node_reached`).

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| name | string | да | Уникальное имя метки |

## Звук (Audio)

### Play SFX

Воспроизводит звуковой эффект.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| sound | string | да | Имя звука или ключ |
| volume | number | нет | Громкость (0–1) |
| pitch | number | нет | Тон (1.0 = нормальный) |
