# Undefscene: Интерфейс

## Экран редактора

Экран делится на три основные зоны:

1. **Левая панель** — палитра нод (перетаскивайте на холст) и кнопки Save / Undo / Redo
2. **Центральный холст** — рабочая область с нодами и связями
3. **Правая панель** — Inspector (настройка выбранной ноды) и информация о проекте

Дополнительные панели (Логи, Yarn Preview и др.) открываются через меню панелей.

## Управление на холсте

| Действие | Как |
|----------|-----|
| Создать ноду | Средняя кнопка мыши по пустому месту (создаёт Dialogue) |
| Добавить ноду из палитры | Перетащите из левой панели на холст |
| Переместиться по холсту | Правая кнопка мыши + движение |
| Выделить несколько нод | Зажать левую кнопку и протянуть |
| Выбрать ноду/ребро | Клик левой кнопкой |
| Удалить ноду/ребро | Правый клик → Delete |
| Соединить ноды | Потянуть от handle одной ноды к другой |
| Задать задержку на ребре | Двойной клик по линии между нодами |
| Условие на ребре | Выбрать ребро → включить Condition в Inspector |

## Inspector

Когда вы выбираете ноду, в правой панели появляются её параметры. Заполните обязательные поля — без них экспорт покажет предупреждение.

Для некоторых нод Inspector показывает дополнительные возможности:

- **Dialogue** — панель Text с Yarn Preview: список нод из `.yarn` файла и текст выбранного блока
- **Follow Path** — Path Preview: форма траектории и анимированный маркер движения

## Visual Editing

**Visual Editing** — отдельное окно с фоном комнаты. Помогает расставить актёров и нарисовать путь движения, видя реальную геометрию комнаты.

Открывается кнопкой на панели. Работает в отдельном окне (видно в Alt+Tab).

### Инструменты Visual Editing

| Инструмент | Что делает |
|-----------|-----------|
| Draw Path | Рисует путь по точкам (кликайте на фоне) |
| Pencil | Свободное рисование пути |
| Eraser | Удаляет точки пути |
| Undo Point | Отменяет последнюю точку |
| Clear Path | Удаляет весь путь |
| Import Path | Сохраняет нарисованный путь в Follow Path ноду |
| Place Selected Actor | Ставит актёра на позицию |
| Import Actors | Сохраняет позиции актёров в Actor Create ноды |
| Select | Выделяет маркер актёра |
| Play | Анимирует актёра вдоль пути (только превью, не сохраняет) |

### Клавиши-модификаторы

- **Shift** — привязка к горизонтали/вертикали
- **Ctrl** — привязка к сетке 20px
- **Shift + Ctrl** — оба режима

!!! tip "Play — это только превью"
    Кнопка Play показывает, как актёр будет двигаться, но **не записывает** результат в граф. Для сохранения используйте Import Path / Import Actors.

## Preferences

Окно настроек: **Edit → Preferences** (или кнопка в левом dock).

### General

- **Language** — `English` или `Русский`
- **Theme** — тема интерфейса
- **Accent Color** — цвет акцентов

### Canvas

- **MiniMap** — вкл/выкл миникарту в углу холста
- **Zoom Speed** — скорость зума колесом мыши
- **Grid Size** — шаг фоновой сетки (визуальный, не влияет на расстояние между нодами)

### Background

Можно поставить свою картинку на фон холста. Настройки: привязка (к холсту или к экрану), режим (Stretch / Cover), прозрачность 0–100%.

### Auto-save

- Вкл/выкл автосохранение
- Интервал автосохранения
- Для новых сцен — черновик в памяти приложения
- Для сохранённых сцен — backup-файлы рядом с основной сценой

### Keyboard Shortcuts

Список всех горячих клавиш с возможностью переназначить.

## Продвинутые инструменты

Технические инструменты спрятаны в **Help → Advanced**:

- **Open DevTools** — инструменты разработчика браузера
- **Runtime JSON** — просмотр JSON-экспорта (полезно для отладки)
- **Disable Hardware Acceleration** — если редактор тормозит
