# Undefscene: Интерфейс и Preferences

## Основные зоны интерфейса

1. **Левая панель** — палитра нод и быстрые действия вроде `Save`, `Undo`, `Redo`.
2. **Центральный холст** — рабочая область, где располагаются ноды и связи.
3. **Правая панель** — Inspector и блок интеграции с проектом GameMaker.
4. **Дополнительные панели** — скрытые и floating panels, которые можно открыть через меню.

## Панели

В интерфейсе используются переведённые русские заголовки основных панелей.

Через меню панелей можно открыть, например:

- `Логи / Предупреждения`
- другие вспомогательные панели

Для `dialogue`-нод панель `Text` умеет показывать `Yarn Preview`: список нод из выбранного `.yarn` файла и содержимое выбранного блока в режиме чтения.

Для `follow_path`-ноды Inspector теперь показывает отдельный `Path Preview`: форму траектории, порядок waypoint-ов и анимированный marker движения.

## Управление на холсте

- **Создание ноды** — средняя кнопка мыши по пустому месту холста.
- **Pan** — правая кнопка мыши и движение мыши.
- **Box Select** — зажать левую кнопку мыши и протянуть по пустому месту.
- **Выделение** — клик по ноде или связи.
- **Удаление** — правый клик по ноде или связи.
- **Соединение нод** — потянуть связь от handle одной ноды к другой.
- **Wait на ребре** — двойной клик по связи открывает настройку ожидания.
- **Edge Condition** — на выбранном ребре можно настроить условия перехода.

## Preferences

В окне Preferences доступны основные настройки редактора.

### General

- **Language** — переключение между `English` и `Русский`.
- **Theme** — выбор темы интерфейса.
- **Accent Color** — акцентный цвет UI.

### Canvas

- **MiniMap** — включение или отключение миникарты.
- **Zoom Speed** — скорость зума колесом мыши или trackpad.
- **Grid Size** — меняет шаг фоновой сетки на canvas. Эта настройка влияет на визуальную сетку, а не на расстояние между новыми нодами.
- **Dock Drop Preview** — подсветка целевого места при перетаскивании панели в dock.

### Background

Можно выбрать собственное изображение для фона холста.

- Файл копируется во внутреннюю папку приложения `userData`.
- Можно выбрать режим привязки: к `canvas` или ко всему экрану редактора (`screen-fixed`).
- Поддерживаются режимы `Stretch` и `Cover (crop)`.
- Доступна отдельная настройка прозрачности `0–100%`.
- Абсолютный путь к выбранной картинке в UI не показывается.

### Auto-save

- Включение или отключение autosave.
- Настройка интервала autosave.
- Для новой сцены используется autosave draft в `userData`.
- Для сохранённой сцены создаются autosave-backup файлы рядом с основной сценой.

### Keyboard Shortcuts

В Preferences показывается актуальный список shortcuts и доступно изменение сочетаний.

## Dock-поведение и UX-правки

- Левый, правый и нижний dock можно сворачивать.
- Нижний dock больше не должен сам раздуваться после сценария `сузить окно -> вернуть размер`.
- Floating-панели при resize корректнее ограничиваются внутри рабочей области.
- Resize окна и drag-resize сплиттеров обновляют layout через `requestAnimationFrame`, чтобы уменьшить лишние перерисовки.
- `Runtime JSON` при resize панели теперь растягивается вместе с доступной высотой, а не остаётся в фиксированном маленьком блоке.

## Advanced / Help

Технические действия перенесены в `Help > Advanced`.

Там доступны:

- `Open DevTools`
- `Runtime JSON`
- `Disable Hardware Acceleration`

Если `Runtime JSON` открыто в dock или floating panel, содержимое JSON теперь занимает доступную высоту панели целиком.
