# Управление в игре

## Горячие клавиши

| Клавиша | Действие | Контекст |
|---------|----------|----------|
| **WASD / Стрелки** | Движение | Игровой процесс |
| **Shift** | Бег | Игровой процесс |
| **Enter / Z** | Подтверждение / взаимодействие | Игровой процесс, меню |
| **Q** | Переключение полного экрана (`borderless`) | В любой момент |
| **↑/↓ / W/S** | Выбор пункта меню | Меню |
| **Escape / X** | Назад / выход | Меню |
| **B** | Удалить сейв | Меню сейвов |

## Debug-режим

Работает только при включённом debug-режиме (`F12 × 5`).

| Клавиша | Действие |
|---------|----------|
| **F1** | Показать / скрыть информацию о позиции и направлении |
| **F2** | Показать / скрыть коллайдеры |
| **F3** | Показать / скрыть хитбокс персонажа / interaction marker |
| **F4** | Полная перезагрузка игры |
| **F5** | Предыдущая комната |
| **F6** | Следующая комната |
| **F7** | Быстрый сейв |
| **F8** | Призрак (движение сквозь препятствия) |
| **F9** | Toggle music overlay (требуется debug mode) |
| **F12 × 5** | Включить / выключить debug mode |
