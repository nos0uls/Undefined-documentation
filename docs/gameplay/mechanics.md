# Игровые Механики (Mechanics)

Этот раздел кратко фиксирует основные игровые механики, которые уже отражены в других документах проекта.

## Текущие механики верхнего уровня

- **Перемещение игрока**: движение, бег и debug-хоткеи описаны в [controls.md](./controls.md).
- **Взаимодействие с миром**: запуск через `confirm`, проверка направления игрока и `obj_pointMarker` описаны в [systems/interaction.md](../systems/interaction.md).
- **Диалоги и Yarn**: логика работы Yarn и связанного runtime описана в системных разделах и документации по катсценам.
- **Катсцены**: катсцены могут временно перехватывать ввод и управлять камерой, игроком и dialogue flow.

## Что пока документировано частично

В проекте есть связанные gameplay-аспекты, которые пока лучше раскрыты через системные документы, а не через отдельную gameplay-страницу:

- UI blocking (`scr_checkUIBlocking`) — блокировка ввода при открытых меню, диалогах и катсценах
- музыка и переходы между состояниями
- катсценная интеграция игрока
- debug-инструменты
